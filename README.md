# Course: Data Infrastructure and Introduction to Data Science (SQL)

## Project Title: Human Resources Data Queries

### Project Overview

In practice, **SQL problems involving human resources (HR) data** are very common. In particular, two entities frequently encountered are **employees** and **departments**.
This project requires applying the SQL querying skills learned in class to complete the given tasks.

Resources:

The department's table contains the following columns:
	•	department_id: Department ID
	•	department_name: Department name
	•	location_id: Location ID of the department

The employees table contains the following columns:
	•	employee_id: Employee ID
	•	first_name: Employee’s first name
	•	last_name: Employee’s last name
	•	email: Employee email address
	•	phone_number: Employee phone number
	•	hire_date: Employee hire date
	•	job_id: Job ID
	•	salary: Salary
	•	commission_pct: Commission percentage
	•	manager_id: Manager ID of the employee
	•	department_id: Department ID where the employee works
